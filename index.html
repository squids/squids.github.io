Squids

An image data extraction framework

Welcome to Squids.

Squids is an image data extraction open source framework. Check out the code at http://github.com/squids/squids-core

Phases

Phase 1

Data extraction from image:

  • EXIF information
    • date (day, month, year)
    • histogram classification
    • place - requires GPS position and 3rd. party systems

Phase 2

  • Organization of photos

Phase 3

  • Use of deep machine learn to improve organization and suggestion.
